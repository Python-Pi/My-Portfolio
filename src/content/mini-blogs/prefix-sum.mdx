## Prefix Sum

Prefix sum involves precomputing the cumulative sums of an array to enable queries regarding subarray sums like range sum.
This allows `O(1)` time complexity for sum queries after an `O(n)` preprocessing step.

### When to Use Prefix Sum?

- **Multiple Range Sum Queries**: When you need to frequently calculate the sum of elements between two indices in an array.

## Template

```cpp
vector<int> prefixSum(n+1, 0);
for(int i = 0; i < n; i++){
    prefixSum[i+1] = prefixSum[i] + arr[i];
}
```

```python 
prefix_sum = [0] * (n + 1)
for i in range(n):
    prefix_sum[i + 1] = prefix_sum[i] + arr[i]
```
